# nBall UI Starter (nDS v1)

A minimal Next.js (TypeScript) starter with **nBall Design System (nDS v1)** tokens wired into **Tailwind** and a small **nUI kit**.

## Run
```bash
npm install
npm run dev
```

Open: http://localhost:3000

## Rules (non-negotiable)
- No hard-coded hex colors in product code (use tokens).
- No custom spacing values; use tokenized Tailwind utilities.
- Use nUI components from `components/nds/` for consistent UI + A11y.
